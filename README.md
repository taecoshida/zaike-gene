## GitHub Pages

1. Create a new GitHub repository named `zaike-gene`.
2. Upload `index.html` and the `images/` folder.
3. Open repository Settings → Pages.
4. Set Source to `Deploy from a branch`.
5. Choose Branch: `main`, Folder: `/root`.
6. Save.
7. Open the published URL after deployment finishes.

## Cloudflare Pages

1. Put this folder into a GitHub repository.
2. In Cloudflare Pages, create a project from that repository.
3. Framework preset: `None`.
4. Build command: leave empty.
5. Output directory: `/` or leave empty depending on UI.
6. Deploy.
